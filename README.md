#### Descrição do projeto:

Um quiz sobre qualquer coisa, pode criar salas dos assuntos, perguntas anônimas e os criadores das salas respondem. 


#### Rodar o projeto

No terminal 

```
npm start
```
Roda localmente na porta: http://127.0.0.1:3000/
